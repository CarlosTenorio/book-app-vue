# book-app-vue

Real world Vue 3 App.

Book Management with Google API

Features:

-   API calls
-   Arquitecture based on modules to scale apps
-   Encapsulated styles
-   Handle async data
-   Multiple environments
-   VUEX 4 (state management for Vue 3)
-   Translations
